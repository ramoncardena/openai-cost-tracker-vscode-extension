# Change Log

All notable changes to the "openai-cost-tracker" extension will be documented in this file.

## [0.0.1] - 2025-12-16

### Added

- **Cost Tracking**: Real-time status bar item showing current day or month cost.
- **Secure Authentication**: Secure storage of OpenAI User API Key.
- **Status Bar Menu**: Quick access to view modes and refresh.
- **Commands**: `setApiKey`, `showCost`.
- **Configuration**: Customizable refresh interval.
